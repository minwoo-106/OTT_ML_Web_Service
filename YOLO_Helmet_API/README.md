\# YOLO Helmet Detection API



2차 프로젝트로 진행한 자전거/오토바이 탑승자 헬멧 착용 여부 탐지 프로젝트입니다.



\## 주요 기능



\- YOLO 기반 헬멧 착용/미착용 탐지

\- 이미지 및 영상 입력 처리

\- Flask 기반 웹/API 테스트

\- 탐지 결과 시각화



\## 참고



모델 가중치 파일(.pt), 업로드 파일, 결과 영상은 용량 문제와 저장소 관리를 위해 GitHub에 포함하지 않았습니다.

로컬 실행 시 필요한 모델 파일은 별도로 `yolo\_models/` 폴더에 배치해야 합니다.
